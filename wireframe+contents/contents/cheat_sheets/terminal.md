## Unix terminal shortcuts:
| Shortcut | Action |
| - | - |
| `ctrl+c` | kill current foreground process |
| `ctrl+z` | suspend current foreground process |
| `ctrl+l` | clears screen |
| `ctrl+a` | move cursor to beginning of line |
| `ctrl+e` | move cursor to end of line |
| `ctrl+u` | erase line |
| `ctrl+k` | erase part of line after cursor |
| `ctrl+w` | erase word before cursor |
| `ctrl+y` | paste previusly erased text |
| `ctrl+r` | search for command in history |
